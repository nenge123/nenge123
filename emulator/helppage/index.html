DraStic安装
先安装
DraStic r2.5.1.2a.apk
VirtualXposed_0.18.2.apk
Jasi_Patcher_v4.4.apk
启动:VirtualXposed
  • 按住菜单键,添加程序,勾选:DraStic,Jasi_Patcher,安装.
  • 按住菜单键,选择模块:Jasi,模拟:勾选所有选项.应用选择:DraStic.
  • 安卓,设置,应用设置. 给VirtualXposed添加快捷方式权限.(可选)
  • VX主界面往推,在里面打开DraStic,(弹框添加快捷方式->是),添加游戏游玩.
IPA安装
第一步 - 绑定阶段
  • 已安装最新爱思助手,itunes(可爱思里面安装),可联网电脑一台,一个苹果账号.
  • 手机连接电脑,点开工具箱→IPA签名→使用Apple ID 签名→添加 Apple ID
  • 输入正确用户名/密码,不需要本机账号,可以自行注册一个新账号.
  • 成功绑定过程中是有双重验证.
  • 不同手机都需要添加一次绑定.(绑定UUID)
第二步 - 签名阶段
  • 添加IPA文件.最多三个.不要分开签名.非要分开最好一个应用一个账号签名.
  • 勾选列表下得APP.勾选账号,开始签名.
  • 如果签名失败.更新软件,如果最近更改了密码,需要重新绑定.
  • 签名成功后,点开储存目录,双击刚刚签名的应用,看时间判断是不是最近的.
  • 签名会生成新文件,不是双击添加时的添加文件.
第三步 - 重装阶段
  • APP应用打不开就是签名失效了.
  • 使用苹果的 文件应用.双击浏览→我的iPhone→失效的模拟器应用→进入文件夹.
  • 把里面的文件夹移动走其他目录,如下载.
  • 删掉APP,重装,运行一次模拟器关掉,再次使用文件应用.打开模拟器目录,删掉里面的文件夹.
  • 把之前移动走的文件夹移动回模拟器目录,运行模拟器.
